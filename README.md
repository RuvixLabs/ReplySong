# ReplySong

Flutter mobile app for Ruvix Labs that turns message replies into AI-generated songs.

The project shell is ready. The Flutter app, Firebase apps, RevenueCat project, and store subscription products still need to be scaffolded/configured.

## Stack

Flutter · Firebase · RevenueCat · AI music generation backend

## Next Setup Steps

1. Scaffold the Flutter app.
2. Create iOS and Android bundle/package identifiers.
3. Create Firebase apps for iOS and Android.
4. Create the RevenueCat project, apps, entitlement, offerings, and store products.
5. Wire generation only after an active RevenueCat entitlement is confirmed.
